# Email Analyzer V4 — Smart Clickable Links

This version captures hidden hyperlinks from Gmail/Outlook clipboard HTML.

Upload these files to the GitHub repository root:
- app.py
- requirements.txt
- render.yaml
- README.md
- .gitignore

Then use Render:
Manual Deploy > Deploy latest commit

Best result:
Open the email, select the complete email content, copy it, and paste directly into the analyzer.
Do not first paste into Notepad, because Notepad removes hidden hyperlinks.
