# Professional AI Email Analyzer V3 — Clickable Links

This update makes item/article/job/product links clickable when a URL is present in the pasted email or can be verified.

Upload these files to the root of your existing GitHub repository:

- app.py
- requirements.txt
- render.yaml
- README.md
- .gitignore

Then in Render:

1. Open your Web Service.
2. Select **Manual Deploy**.
3. Select **Deploy latest commit**.

Your existing `GEMINI_API_KEY` stays saved in Render.
